Least‑Squares Fitting

1. Purpose

Least‑squares fitting estimates the parameters of a model that best describes data containing noise. By minimising the sum of squared residuals \( S = \sum (y_i - f(x_i;\,\theta))^2 \), the method finds the parameter set \(\theta\) that brings the model curve closest to the data points.

2. Linear Model Example

For a straight‑line model \( y = mx + b \), the optimal slope and intercept are found analytically or via numerical routines (e.g. numpy.polyfit). The residual for each point is the vertical distance between the datum and the fitted line.

3. Error Metrics

RMSE (root‑mean‑square error) summarises residual magnitudes: \( \text{RMSE}=\sqrt{\langle r^2 \rangle}\). R² (coefficient of determination) measures how much variance the model explains: \( R^2 = 1 - S/S_{\text{tot}} \).
