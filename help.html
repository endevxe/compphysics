Help & Documentation

Frequently Asked Questions (FAQ)

What is Episteme? Episteme is a student-created platform that offers open-source physics simulations to help students and enthusiasts explore theoretical models, numerical solutions, and visualizations.
Do I need to install anything to run the simulations? Most simulations are written in Python and require Python 3 and packages like matplotlib, numpy, or scipy. Some are fully interactive in-browser using JavaScript.
Can I suggest a simulation or contribute? Absolutely! Head over to the Contact page and send in your idea — whether it's a concept, an improvement, or a bug fix.
Are these simulations accurate? All simulations are built with the intention to reflect ideal theoretical or numerical models. Some are simplified or scaled for visualization and learning purposes.
Is this content free to use? Yes, everything on Episteme is open-source and free for educational or personal use.

How to Use the Source Code

Each project page includes a download link or in-browser code viewer. You can:

  • Download the .py file and open it in a Python IDE (like Thonny, VSCode, or PyCharm).
  • Ensure you have Python 3 installed on your machine. If not, download it from python.org.
  • Install required libraries using pip. Example: pip install numpy matplotlib
  • Run the script in your terminal or IDE. Most simulations will generate visualizations or animation windows.

For browser-based simulations, simply click the "View Simulation" button. These do not require downloads or installation.
