Help & Documentation

Frequently Asked Questions (FAQ)

What is Episteme? Episteme is an open‑source hub of physics simulations for students and enthusiasts.
Do I need to install anything? Most simulations run with Python 3 and libraries like numpy & matplotlib, or directly in your browser.
Can I suggest a simulation? Yes! Use the Contact page to share ideas or improvements.
Is the content free to use? All material is free for educational and personal use under an open‑source licence.

How to Use the Source Code

Each project page links to the code. Typical workflow:

  • Download the .py file.
  • Install requirements: pip install numpy matplotlib
  • Run from the command line: python simulation.py
  • Browser‑based projects run instantly – just click “View Simulation”.

Command‑Line Help

Open a terminal in the simulation folder and type:

python simulation.py --help

Many scripts accept flags (e.g. --steps 5000) so you can tweak parameters.
